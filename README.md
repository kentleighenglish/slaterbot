# Slaterbot

Tracks and trades stock using the Alpaca API and data from various APIs

## Setup

Install dependencies
```bash
# yarn
yarn install

# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
